# 说明
由于 **The Most Forbidden Love in the World** 2019 (即英文无码版)  
有部分CG为动态（实际上就是一段视频）  
所以适配的 **世界でいちばんNG（だめ）な恋** 2007版本  
有13张CG仍然为有码（由于视频色彩和画质稀烂，暂不截图替换）  

由于 **世界でいちばんNG（だめ）な恋** 2007版本  
游戏引擎似乎不带有Patch封包的功能  
所以直接替换了整个Chip.arc来实现替换无码
